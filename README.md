## 🌍 Overview

**AutoML Studio** is a sleek and powerful **no-code machine learning platform** that allows users to:

✔ Upload dataset  
✔ Analyze data instantly  
✔ Generate visual insights  
✔ Train multiple ML models automatically  
✔ Download the best model  

💡 Built for students, beginners, and rapid prototyping.

---

## 🎯 Key Highlights

- ⚡ End-to-end ML pipeline  
- 🎨 Beautiful dark-themed UI (custom CSS)  
- 🧠 Smart model selection using PyCaret  
- 📊 One-click data profiling report  
- 💾 Export trained model instantly

## ⚙️ Installation

```bash
git clone https://github.com/abhishek052005/AutoML-Studio.git
cd AutoML-Studio
pip install -r requirements.txt
```
---
## 🧠 How It Works

1. Upload dataset (CSV)
2. System analyzes structure and statistics
3. Generates visualization report
4. Trains multiple ML models using PyCaret
5. Compares models and selects best one
6. Export best model as .pkl file
